# 压力过载系统设计 (Stress Overload System)

## 设计灵感

本系统参考 **Darkest Dungeon** 的压力/美德/ affliction 机制，将其适配到狼人杀推理游戏的社交压力场景中。

在 Darkest Dungeon 中：
- 英雄压力值 0-200，100 时触发 **Resolve Test**（75% 获得 affliction，25% 获得 virtue）
- 200 时触发 **Heart Attack**（死亡或濒死）
- Affliction（ affliction）使英雄失控：拒绝指令、攻击队友、增加全队压力
- Virtue（美德）使英雄超常发挥：属性提升、减少全队压力、自我恢复

## 适配设计

### 基本规则

| 参数 | 值 | 说明 |
|------|-----|------|
| 压力范围 | -10 ~ +10 | 已有设计 |
| 过载阈值 | **+10** | 压力达到上限时触发过载测试 |
| 测试机制 | Resolve Test | 决定获得美德还是 affliction |
| 持续时间 | 当前白天阶段 | 效果持续到阶段结束（不同于DD的整局） |

### Resolve Test (过载测试)

当角色压力达到 +10 时，立即触发 Resolve Test：

```
美德概率 = 基础 25% + 亲和修正 + 阵营修正 + 历史修正
 affliction概率 = 1 - 美德概率
```

#### 修正因素

| 因素 | 对美德概率的影响 | 说明 |
|------|------------------|------|
| 基础概率 | +25% | 默认 |
| 高亲和 (>7) | +10% | 情绪稳定者更容易保持理智 |
| 守序善良 | +10% | 道德坚定者抗压能力强 |
| 混乱邪恶 | -15% | 性格阴暗者更容易崩溃 |
| 本局已获得 affliction | -10% | 重复崩溃递减 |
| 本局已获得 virtue | -20% | 已获得美德者难以再次触发 |
| 被多人怀疑 | -5% × 怀疑人数 | 社交压力降低美德概率 |

### Virtue (美德) - 压力下的英雄时刻

美德是角色在极端压力下迸发出的超常状态。角色获得美德后，立即执行一次积极行为（如强烈指认狼人、担保好人、号召投票等），且行为效果大幅增强。

| 美德类型 | 效果 | 适用角色 |
|----------|------|----------|
| **坚定 (Stalwart)** | 本阶段所有检定 +3，免疫压力增加 | 通用 |
| **无畏 (Fearless)** | 强烈指认/担保时，旁观者信任 +2（额外） | 村民 |
| **洞察 (Insightful)** | 观察成功率 +50%，获取额外信息 | 预言家/神职 |
| **团结 (Rallying)** | 所有队友压力 -2，信任 +1 | 通用 |
| **复仇 (Vengeful)** | 被攻击/怀疑时，反击效果翻倍 | 通用 |

#### 美德特性
- 触发美德时，角色压力立即降至 +5
- 美德角色的行动会触发额外日志（金色/白色特效提示）
- 美德有传染性：旁观者见证美德后，自身美德概率 +5%（本轮限一次）

### Affliction (崩溃/负面状态)

 affliction 是角色在极端压力下心理崩溃的表现。角色进入 affliction 状态后，其行动可能偏离 AI 原本意图，甚至做出对己方不利的行为。

| affliction类型 | 效果 | 适用角色 |
|----------------|------|----------|
| **恐惧 (Fearful)** | 强制沉默 1-2 轮，跳过所有行动 | 通用 |
| **偏执 (Paranoid)** | 随机怀疑盟友/信任敌人，关系判断反转 | 通用 |
| **自私 (Selfish)** | 只执行保护自己的行为，拒绝帮助他人 | 通用 |
| **好斗 (Abusive)** | 随机攻击（怀疑/指认）任意目标，包括可能的队友 | 通用 |
| **绝望 (Hopeless)** | 强制弃票，认为"无所谓了" | 通用 |
| **非理性 (Irrational)** | 行动完全随机，无视策略和目标 | 通用 |
| **狂躁 (Rapturous)** | 过度暴露，公布过多信息（如真实身份） | 通用 |
| **受虐 (Masochistic)** | 主动暴露自己，吸引怀疑/攻击 | 通用 |

#### affliction 特性
- 触发 affliction 时，角色压力降至 +7（仍然很高）
- affliction 角色的异常行动会触发额外日志（红色/暗色特效提示）
- affliction 有传染性：旁观者见证 affliction 后，压力 +1（本轮限一次）
- 同一角色一天内最多触发一次 Resolve Test

### 与现有系统的关系

#### 与阵营的交互

| 阵营 | 美德倾向 | affliction 倾向 |
|------|----------|-----------------|
| 守序善良 | 高 | 低（崩溃后更可能表现为沉默/自我牺牲） |
| 守序邪恶 | 中 | 中（崩溃后可能攻击他人） |
| 混乱善良 | 中 | 中（崩溃后可能非理性行动） |
| 混乱邪恶 | 低 | 高（崩溃后可能攻击队友/暴露身份） |

#### 与关系的交互

- 见证好友获得美德：信任 +1，压力 -1
- 见证好友获得 affliction：压力 +2，信任不变（同情）
- 见证敌人获得美德：压力 +1，怀疑加重
- 见证敌人获得 affliction：信任 +0.5（幸灾乐祸的微妙信任）

#### 与属性的交互

| 属性 | 对美德/affliction 的影响 |
|------|---------------------------|
| 亲和 | 高亲和提升美德概率，降低传染压力 |
| 逻辑 | 高逻辑使 affliction 时更可能表现为偏执而非非理性 |
| 洞察 | 高洞察使美德时更容易获得"洞察"美德 |
| 隐蔽 | 高隐蔽使 affliction 时更可能隐藏崩溃（外部不明显） |
| 领导 | 高领导使美德时更容易获得"团结"美德 |
| 诡诈 | 高诡诈使 affliction 时更可能表现为攻击/暴露而非沉默 |

### 设计目的

1. **增加戏剧性**：压力过载不只是一个"debuff"，而是角色命运的转折点
2. **增加不可预测性**：即使是最理性的 AI，在极端压力下也可能做出出乎意料的行为
3. **丰富叙事**：美德和 affliction 都是强叙事元素，让日志更有故事感
4. **平衡高风险行为**：玩家/AI 可以选择是否将角色逼入压力过载，但需承担不可控后果
5. **呼应 Darkest Dungeon 主题**：承认人类在压力下可能崩溃，也可能爆发出英雄时刻

> 参考：Darkest Dungeon 的 Affliction 系统被设计为"打破玩家控制"的机制，这正是我们需要的——在狼人杀中，没有人能在极端社交压力下保持完全理性。
