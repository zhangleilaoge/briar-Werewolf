# 狼人杀特殊角色参考表

本文档汇总常见狼人杀变体中**除普通村民、普通狼人以外**的特殊角色及其效果，供本项目设计角色能力时参考。

> 来源：狼人杀官方版型、网易狼人杀、各民间变体与百科整理。不同版本规则可能存在差异，以下效果为常见描述。

---

## 好人阵营（神职 / 特殊平民）

| 角色 | 类型 | 技能效果 | 备注 |
|------|------|----------|------|
| 预言家 | 信息 | 每晚查验一名存活玩家是好人还是狼人 | 通常仅知阵营，不知具体身份 |
| 女巫 | 攻防 | 拥有一瓶解药和一瓶毒药；解药可救活当晚被狼刀玩家，毒药可毒杀一名玩家；每晚最多用一瓶 | 多数规则不可自救；用过解药后通常看不到刀口 |
| 猎人 | 进攻 | 被狼刀或被放逐出局时可开枪带走一名玩家；被毒杀通常不能开枪 | 可选择压枪不发动 |
| 守卫 | 防御 | 每晚守护一名玩家免疫狼刀，可自守，不能连续两晚守护同一人 | 与女巫同守同救可能导致目标死亡（奶穿） |
| 白痴 / 愚者 | 防御 | 白天被放逐时可翻牌免疫此次放逐，之后失去投票权但可继续发言 | 狼人通常需要追刀 |
| 骑士 | 进攻 | 白天发言阶段可随时翻牌，选择一名玩家决斗；对方是狼人则对方立即出局并进入黑夜；是好人则骑士出局 | 一次性技能 |
| 禁言长老 | 控制 | 每晚指定一名玩家，下一天白天发言阶段被禁言 | 只能以手势表达 |
| 禁票长老 | 控制 | 每晚指定一名玩家，下一天白天不能投票 | — |
| 狐狸 | 信息 | 每晚查验一名玩家及其左右相邻玩家中是否有狼人；若三人皆好人，狐狸失去技能 | 不知具体谁是狼 |
| 子狐 | 控制/信息 | 从第二晚开始可迷惑一名玩家；若迷惑到狼人，当晚狼队不能刀人 | 一次性技能 |
| 驯熊师 / 熊 | 信息 | 天亮后若相邻存活玩家中有狼人，法官宣布“熊咆哮了”；否则“熊没有咆哮” | 只侦查左右相邻 |
| 守墓人 | 信息 | 从第二晚起得知上一白天被放逐玩家的阵营 | — |
| 摄梦人 | 防御/进攻 | 每晚必须选择一名“梦游者”，梦游者免疫夜间伤害；摄梦人夜间死亡则梦游者一并死亡；连续两晚梦游同一人则其死亡 | 不可连续两晚梦游同一人 |
| 觉醒摄梦人 | 信息/防御 | 每晚选择一名“梦语者”，梦语者免疫夜间伤害；觉醒摄梦人得知梦语者是否行动，从第二晚起可选择是否将其梦出局 | 不可连续两晚选择同一人 |
| 乌鸦 | 控制 | 每晚诅咒一名玩家，下一天放逐时该玩家额外被投一票 | 不能连续两晚诅咒同一人 |
| 魔术师 | 控制 | 每晚交换两名玩家的号码牌，当晚所有指向这两人的技能目标互换 | 每个号码牌整局只能交换一次 |
| 猎魔人 | 进攻 | 从第二晚起每晚狩猎一名玩家；目标是狼人则狼人出局；目标是好人则猎魔人出局 | 可多次发动 |
| 纯白之女 | 信息/进攻 | 每晚查验一名玩家；从第二晚起，查验到狼人则该狼人直接死亡 | 被狼巫查验会直接死亡 |
| 定序王子 | 控制 | 每局限一次，在第一次放逐投票结束后可翻牌逆转时空，回到投票前重新投票 | 重新投票前可额外发言 |
| 奇迹商人 / 黑市商人 | 辅助 | 每局限一次，选择一名玩家赋予额外技能（预言家查验、女巫毒、守卫守护/猎人枪）；若目标是狼人，商人次日死亡 | 目标可能不知道自己被选中 |
| 老流氓 | 防御 | 不受狼美人魅惑；被毒杀或射杀后进入中毒/负伤状态，第二天发言结束后才死亡 | 延迟死亡 |
| 替死鬼 | 防御 | 放逐投票平票时，替死鬼被放逐，并可指定哪些玩家可参与下一天投票 | 替死鬼出局 |
| 河豚 | 防御/进攻 | 被放逐后可翻牌带走所有投自己的人；或狼刀河豚后狼美人白天技能失效 | 一次性 |
| 白猫 | 防御 | 任何原因出局时可翻牌，额外存活到下次放逐投票后 | — |
| 炼金魔女 | 攻防 | 拥有“未明之雾”（指定3人，狼人当晚只能刀这3人）和“法老之蛇”（救活被刀玩家）；各限一次 | 法老之蛇未使用时，白天先发言后公布死亡 |
| 白昼学者 | 辅助/控制 | 从第二晚起可增幅或削弱一名角色各一次；增幅使目标当夜技能次数+1，削弱使目标当夜无法使用技能 | 与寂夜导师同目标好人则该好人出局；同目标狼人则寂夜导师出局 |
| 流光伯爵 | 防御 | 每晚庇护一名玩家，被庇护者当夜不受任何技能影响 | — |
| 通灵师 | 信息 | 每晚查验一名玩家的真实身份 | 可识别机械狼复制后的身份 |
| 术士 | 信息 | 第一晚指定一名玩家，若该玩家当晚死亡，术士得知伤害来源；被狼杀则得知最近狼人 | — |
| 灵能者 | 信息 | 每晚最后睁眼，得知当晚所有成为技能目标的玩家，但不知具体被谁选中 | 适合大局 |
| 时序者 | 辅助 | 标记一名玩家，其死亡时预警；可给予一名玩家额外一天存活时间 | 拥有3天时间资源 |
| 觉醒愚者 | 防御 | 拥有“秘密之身”，可保护一名玩家抵消一次夜间伤害；被放逐时若有秘密之身可翻牌免疫 | 成功抵消后失去秘密之身 |
| 觉醒预言家 | 信息 | 每晚选择两名玩家，判断其中是否有狼人 | 不知具体谁是狼 |
| 魔镜少女 | 信息 | 类似预言家，可查验阵营 | 不同版本效果略有差异 |
| 熊 / 驯熊师 | 信息 | 见上文“驯熊师” | — |
| 面包师 / 占卜师 | 防御 | 每晚标记一名玩家，狼人当晚必须攻击该玩家或其左右相邻玩家，且不得空刀 | 一次性 |
| 圣女 | 控制 | 预言家之后睁眼，选择一名玩家净化；若是狼人，该狼人永久失去夜间杀人能力 | 一次性 |
| 神父 | 进攻/防御 | 其他神职全部出局前免疫夜间伤害；可使用一次性“圣光”，对吸血鬼（狼人方）直接出局 | 神职全灭后不再免疫 |
| 女巫的学徒 | 辅助 | 女巫死后获得制毒/制药能力，但只能选其一且发动一次；由上帝掷硬币决定是否成功，失败则自己死亡 | — |

## 狼人阵营（特殊狼人）

| 角色 | 类型 | 技能效果 | 备注 |
|------|------|----------|------|
| 狼王 | 进攻 | 死亡时（除被毒、殉情外）可开枪带走一名玩家 | 自爆通常不能带人 |
| 白狼王 | 进攻 | 白天自爆时可带走一名玩家，随后进入黑夜 | 被毒或骑士决斗不能发动 |
| 黑狼王 | 进攻 | 与狼王类似，死亡时可开枪带走一人；被毒不能发动 | 部分版本等同于狼王 |
| 狼美人 | 控制/进攻 | 每晚魅惑一名玩家；狼美人死亡时，前一晚被魅惑的玩家殉情且不能发动技能 | 不能自爆、自刀；不能连续两晚魅惑同一人 |
| 隐狼 | 伪装 | 不与狼队一起睁眼刀人，狼队友不知其身份；被预言家/狐狸/骑士判定为好人；其他狼全灭后可带刀 | 不能自爆；部分版本狼队友可刀隐狼 |
| 雪狼 | 伪装 | 与狼队一起刀人；被预言家/狐狸判定为好人；与骑士决斗则死亡 | 隐狼改版 |
| 石像鬼 | 信息 | 不与狼队一起睁眼，每晚查验一名玩家的具体身份；其他狼全灭后可带刀 | 不能自爆；知道所有狼人 |
| 觉醒石像鬼 | 信息/进攻 | 石像鬼强化版，具体规则依版本而定 | — |
| 梦魇 / 噩梦之影 | 控制 | 每晚在其他玩家行动前恐惧一名玩家，被恐惧者当夜无法使用技能；恐惧到狼人则狼队当晚空刀 | 不能连续两晚恐惧同一人 |
| 恶灵骑士 | 防御/进攻 | 一次性反伤：被预言家查验或女巫毒杀时，发动者死亡；夜间无法死亡 | 不能自爆、自刀；骑士决斗可杀 |
| 血月使徒 | 控制/进攻 | 自爆后当晚所有好人技能被封印；作为最后一张狼人被放逐时，可存活到下一晚再刀一人，次日白天出局 | — |
| 机械狼 | 信息/辅助 | 不与狼队一起睁眼，有一次学习他人技能的机会；学习后可使用该技能；被通灵师查验显示为所学身份 | 其他狼全灭后可带刀；不能自爆 |
| 种狼 | 控制 | 一次性技能：在狼人刀人后额外睁眼，可选择是否让被刀且未被救/守的玩家变为狼人 | 被感染者保留原技能 |
| 大野狼 | 进攻 | 与狼队共同刀人后，单独睁眼再刀一人；场上有狼人死亡后不再单独睁眼 | — |
| 毒狼 | 防御/进攻 | 被除放逐外的任意方式死亡后，令杀死自己的玩家中毒失去技能 | — |
| 狼巫 | 信息/进攻 | 每晚查验一名非狼玩家的具体身份；从第二夜起验到纯白之女则纯白之女死亡 | — |
| 狼鸦之爪 | 进攻 | 开局被封印，不知狼队友且不参与刀人；存活狼人少于3人时解封，获得一次性额外刀人技能（无视防御） | 其他狼人知道其身份 |
| 蚀时狼妃 | 控制 | 每晚率先睁眼，将一名玩家封锁到时间夹缝；当晚好人对其使用的验人、毒药、守护技能视为对自身释放 | 技能生效后下一夜永久失去技能；不能自爆、自刀 |
| 寂夜导师 | 辅助/控制 | 与白昼学者对应，可增幅或削弱一名角色各一次；效果同白昼学者但阵营为狼 | 同目标好人出局；同目标狼人则自己出局 |
| 蚀日侍女 | 信息/进攻 | 每晚选择一名非狼人盗取技能；不可重复盗取；其他狼全灭后可杀人 | 偷流光伯爵庇护目标会导致双死 |
| 恶魔 | 信息/防御 | 与狼人共同杀人后，单独睁眼确认一名玩家是否神职；不能死于夜间；不能自爆、自刀 | — |
| 狼外婆 | 辅助 | 部分版本中存在，具体效果依村规而定 | 民间变体 |
| 暴行 / 狼牙之爪 | 进攻 | 民间变体中的进攻型特殊狼人 | 效果依版本而定 |
| 黑无常 / 白无常 | 特殊 | 部分民间变体角色 | 效果依版本而定 |

## 第三方 / 变动阵营

| 角色 | 类型 | 技能效果 | 备注 |
|------|------|----------|------|
| 丘比特 | 变动 | 首夜选择两名玩家成为情侣；情侣同阵营则丘比特归属该阵营；一人一狼则三者组成独立第三方，需屠城获胜 | 情侣一方死亡另一方殉情 |
| 吹笛者 | 第三方 | 每晚迷惑1~2名玩家；当所有存活玩家（除自己）均被迷惑时单独获胜 | 通常不能与丘比特同局 |
| 混血儿 | 变动 | 首夜选择一名玩家作为主人/榜样，胜利条件与主人阵营一致；主人死亡后部分版本变回好人 | 不知主人具体身份 |
| 野孩子 | 变动 | 首夜选择一名榜样；榜样死亡后野孩子变为带刀狼人 | 榜样不知自己被选中 |
| 影子 | 变动 | 选择一名榜样；榜样死亡后影子继承其身份与阵营 | — |
| 暗恋者 | 变动 | 首夜暗恋一名玩家，胜利条件与该玩家阵营一致 | 被暗恋者不知情 |
| 盗贼 | 第三方/变动 | 游戏开始时从两张额外身份牌中选择一张作为自己身份；若两张中有狼人则必须选狼人 | 另一张作废 |
| 咒狐 | 第三方 | 被狼刀不会死亡；被预言家查验则出局；当好人或狼人达成胜利条件时，若咒狐存活则咒狐取代其获胜 | — |
| 炸弹人 | 第三方 | 被投票放逐时爆炸，带走所有投自己的人 | 一次性 |
| 复仇者 | 第三方 | 部分版本中存在，拥有特定复仇目标 | 效果依版本而定 |
| 盗宝大师 | 第三方 | 部分版本中存在，可盗取他人技能 | 效果依版本而定 |
| 鬼魂新娘 | 第三方 | 类似丘比特的情侣变体 | 效果依版本而定 |
| 老爷爷 | 第三方 | 部分民间变体角色 | 效果依版本而定 |

## 附加身份 / 特殊机制

| 角色/机制 | 效果 |
|-----------|------|
| 警长 | 白天投票时拥有1.5票；通常最后发言；可决定发言顺序；警长死亡可移交警徽 |
| 狼人自爆 | 白天翻牌自杀，中断当前流程，直接进入黑夜 |
| 空刀 | 狼人选择当晚不杀人 |
| 自刀 | 狼人选择杀死自己或狼队友 |
| 屠边 / 屠城 | 狼人胜利条件：屠边=杀死所有平民或所有神职；屠城=杀死所有好人 |

## 使用说明

- 本表为设计参考，**不意味着本项目需要全部实现**。
- 在设计本项目角色时，建议优先挑选与“白天数值 + 骰子检定”机制契合的角色进行改编。
- 所有夜晚技能、查验、守护、毒杀等机制，若与本项目设计冲突，应以本项目 `main.md`、`flow.md`、`numeric.md`、`alignment.md` 的核心规则为准。
