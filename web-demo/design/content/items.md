# 道具

## 基本概念

- **道具**是角色拥有的物品，可以启用特定行动、提供检定加值或改变某些规则。
- 道具与属性、特质不同：属性是角色固有能力，特质是角色特性规则，而道具是外部物品，可能获得、失去或转移。
- 所有道具默认为**隐藏道具**，其他玩家需通过观察或特殊能力才能发现。
- **道具是通用的**：任何玩家持有道具都能使用其能力，与职业无关。
- 道具都有**两种功效**：根据持有者的阵营（狼人阵营 / 人类阵营）触发不同效果。
- 每个角色最多拥有 **3 个道具栏位**。获得新道具时若栏位已满，需先丢弃或转移一个已有道具。

## 核心设计理念

### 道具与职业分离

- **道具提供能力，职业只是起点**：预言家 = 村民 + 水晶球，窃贼 = 村民 + 小偷手套
- **任何人可以使用任何道具**：狼人偷到水晶球也能查验，村民捡到尖牙利爪也能反击
- **失去道具 = 失去能力**：预言家失去水晶球后无法查验，但仍是预言家职业
- **职业决定初始道具**：不同职业开局自带不同道具，但道具可以被偷取、交换、掉落

### 示例

- 预言家开局自带水晶球 → 可以查验
- 狼人偷走水晶球 → 狼人可以查验（查验到狼人时水晶球碎裂）
- 预言家失去水晶球 → 无法查验，但仍可执行其他白天行动
- 村民捡到小偷手套 → 可以偷取一次道具

## 道具类型

| 类型 | 说明 |
|------|------|
| 行动前提型 | 拥有该道具才能执行特定行动 |
| 检定加值型 | 在特定检定中提供加值或减值 |
| 一次性消耗型 | 使用一次后消失 |
| 被动效果型 | 持续生效，无需主动使用 |

## 道具示例

| 道具 | 类型 | 狼人阵营功效 | 人类阵营功效 |
|------|------|--------------|--------------|
| 尖牙利爪 | 行动前提型 | 拥有时可在夜晚执行一对一杀戮 | 被攻击时可选择与攻击者同归于尽 |
| 水晶球 | 行动前提型 | 持有时可执行夜间查验；若查验到狼人，水晶球碎裂损坏 | 持有时可执行夜间查验；若查验到狼人，水晶球碎裂损坏 |
| 小偷手套 | 行动前提型 | 持有时可执行一次偷取；使用后损坏 | 持有时可执行一次偷取；使用后损坏 |
| 验尸工具 | 一次性消耗型 | 持有时可执行一次尸检，查看一名死亡角色的所有道具；使用后损坏 | 持有时可执行一次尸检，查看一名死亡角色的所有道具；使用后损坏 |
| 护身符 | 一次性消耗型 | 抵挡一次夜晚杀戮，使用后损坏 | 抵挡一次夜晚杀戮，使用后损坏 |
| 双刃剑 | 一次性消耗型 | 狂狼持有时可与一名玩家同归于尽，并触发平安夜 | 无效果 |

> 其他道具将在后续设计中加入。

## 道具与角色

- **职业决定初始道具**：预言家开局自带水晶球，窃贼开局自带小偷手套，验尸官开局自带验尸工具
- **道具可以转移**：通过偷窃、交换、掉落等方式，道具可以在玩家间转移
- **能力来自道具，不是职业**：预言家的查验能力来自水晶球，不是来自"预言家"职业
- **职业仍是职业**：失去道具不会改变职业身份，预言家失去水晶球仍是预言家（只是无法查验）
- **阵营影响效果**：同一道具在狼人手中与人类手中可能触发不同功效（如尖牙利爪）

## 道具与行动

- 若某行动需要特定道具作为前提，角色没有该道具时无法执行该行动。
- **任何人持有道具都能使用**：不检查职业，只检查是否持有道具
- 道具的检定加值可以与属性、特质、阵营修正叠加，但需遵循数值上限规则。

## 默认道具配置

| 职业 | 默认道具 | 说明 |
|------|----------|------|
| 普通狼人 | 尖牙利爪 | 可在夜晚执行杀戮 |
| 孤狼 | 尖牙利爪 | 独立选择杀戮目标 |
| 狂狼 | 尖牙利爪 + 双刃剑 | 白天可同归于尽 |
| 普通村民 | 无 | 无特殊能力 |
| 预言家 | 水晶球 | 可执行夜间查验 |
| 窃贼 | 小偷手套 | 可偷取一次道具 |
| 验尸官 | 验尸工具 | 可验尸查看道具 |
