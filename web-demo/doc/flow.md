# 对局流程

本文档记录游戏从开局到结束的完整流程，以及每个阶段可进行的行动。

## 回合结构

游戏由多个回合组成，每个回合分为两个阶段：

1. **白天阶段**：讨论、行动、投票放逐。
2. **夜晚阶段**：角色能力、阵营行动与杀戮结算（独立于属性与关系）。

回合循环进行，直到某一方达成胜利条件。

## 白天阶段

### 流程

1. **早晨事件结算**
   - 公布昨夜死亡信息（若有）。
   - 结算 overnight 状态变化。
2. **公开讨论**
   - 所有存活玩家依次发言。
   - 领导属性影响发言权重与议程主导权。
3. **私下交流**
   - 玩家之间可进行一对一私聊。
   - 信任值决定信息共享的深度。
4. **白天行动**
   - 玩家可执行调查、说服、结盟等行动。
   - 所有行动检定使用属性 + 骰子。
5. **投票放逐**
   - 所有存活玩家投票选择一名放逐目标。
   - 领导与关系值影响跟票倾向。
6. **投票结果结算**
   - 票数最高者被放逐。
   - 结算被放逐者与投票者的关系变化。

### 白天可用行动

| 行动 | 主要属性 | 说明 |
|------|----------|------|
| 说服 | 亲和 | 改变他人投票意向或立场 |
| 推理 | 逻辑 | 分析线索并提出公开推论 |
| 识破谎言 | 逻辑 vs 诡诈 | 质疑他人言论的真实性 |
| 撒谎 | 诡诈 | 传播虚假信息 |
| 制造假线索 | 诡诈 | 伪造证据误导他人 |
| 隐藏立场 | 隐蔽 | 隐藏真实阵营与投票倾向 |
| 调查 | 逻辑 | 从场景或物品中获取信息 |
| 结盟 | 亲和 / 友好值 | 与其他玩家建立临时合作关系 |
| 辩护 | 亲和 / 领导 | 为被怀疑对象辩护，影响投票 |

## 夜晚阶段

夜晚阶段由角色能力、阵营规则或其他独立机制决定，**不受五项属性与角色关系影响**。
