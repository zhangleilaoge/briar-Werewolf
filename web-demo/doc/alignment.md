# 阵营九宫格

本文档记录角色阵营（Alignment）的设计，以及阵营对白天行动、关系变化、压力值与胜利条件的影响。

阵营基于两个维度：

- **守序（Lawful）— 混乱（Chaotic）**：对规则、承诺、秩序与传统的态度。
- **善良（Good）— 邪恶（Evil）**：利他/集体 vs 利己/损人利己的倾向。

交叉形成经典九宫格：

| | 善良 | 中立 | 邪恶 |
|---|---|---|---|
| **守序** | 守序善良 | 守序中立 | 守序邪恶 |
| **中立** | 中立善良 | 绝对中立 | 中立邪恶 |
| **混乱** | 混乱善良 | 混乱中立 | 混乱邪恶 |

## 定位

- 阵营是**行为倾向标签**，不是身份阵营（狼人/村民）的替代。
- 阵营只影响**白天阶段**，不影响夜晚杀戮。
- 阵营可被隐藏，其他角色需通过言行与检定推断。

## 阵营修正

### 守序—混乱维度

| 阵营 | 正面效果 | 负面效果 |
|------|----------|----------|
| 守序 | 发起/支持规则性行动时领导 +1；长期守承诺额外积累信任 | 违反承诺信任额外 -2；撒谎时诡诈检定难度 +2 |
| 中立 | 无极端加成，也无极端惩罚 | — |
| 混乱 | 违反承诺/变票/造假时自身惩罚较轻；撒谎时诡诈检定难度 -2 | 他人对混乱角色信任自然恢复更慢，上限 -2 |

### 善良—邪恶维度

| 阵营 | 正面效果 | 负面效果 |
|------|----------|----------|
| 善良 | 辩护/分享信息时亲和 +1；被保护时信任额外 +1 | 被背叛后友好值下降 +50%；陷害无辜时行动效果可能打折 |
| 中立 | 无极端加成，也无极端惩罚 | — |
| 邪恶 | 陷害/误导成功时对方信任额外 -1；转移嫌疑时诡诈检定难度 -1 | 被证实邪恶行径后信任额外 -2；背叛时自身友好惩罚 -1（下限 0） |

## 阵营组合特征

| 阵营 | 关键词 | 白天行动特征 |
|---|---|---|
| 守序善良 | 正义骑士 | 维护秩序、保护弱者，撒谎陷害弱，信任积累快。 |
| 中立善良 | 好心路人 | 乐于助人、愿意分享，但不执着于规则。 |
| 混乱善良 | 叛逆英雄 | 为善可以不择手段，善于制造混乱但目的善意。 |
| 守序中立 | 规则执行者 | 严格按规则行动，重视程序正义。 |
| 绝对中立 | 旁观者 | 高度自保，适合个人胜利条件。 |
| 混乱中立 | 自由人 | 行为不可预测，以自身兴趣为先，结盟短暂。 |
| 守序邪恶 | 暴君 | 利用规则与秩序服务个人利益，擅长合法陷害。 |
| 中立邪恶 | 利己主义者 | 不择手段达成目标，不主动破坏也不维护秩序。 |
| 混乱邪恶 | 破坏者 | 以混乱和伤害为乐，撒谎背叛最强，但也最易被孤立。 |

## 阵营与关系变化

阵营会修正白天事件导致的关系值变化：

| 事件 | 守序 | 混乱 | 善良 | 邪恶 |
|---|---|---|---|---|
| 遵守承诺 | 信任 +1 | — | 友好 +1 | — |
| 违反承诺 | 信任 -2 | 仅 -1 | 友好 -2 | 自身惩罚 -1 |
| 被证实撒谎 | 信任 -1 | — | 友好 -1 | — |
| 陷害无辜成功 | 自身信任 -2 | — | 效果 -50% | 对方信任 -1 |
| 保护/辩护他人 | 友好 +1 | — | 信任 +1 | — |

额外修正每天对同一对象上限 ±3。

## 阵营与压力值

| 阵营 | 压力相关效果 |
|------|--------------|
| 守序 | 高压时隐蔽检定 -1 |
| 混乱 | 高压时诡诈检定波动 ±1 |
| 善良 | 降低他人压力的效果 +1 |
| 邪恶 | 成功陷害他人时自身压力 -1 |
| 中立 | 无修正 |

额外修正单次上限 ±2。

## 阵营与胜利条件

阵营不直接决定身份阵营，但可引入次要胜利条件或个人目标：

- **善良**：希望好人阵营正义获胜。
- **邪恶**：希望混乱或控制局面，可能暗中帮助狼人。
- **守序**：希望维持秩序与公平过程。
- **混乱**：希望局面充满变数，可能破坏稳定节奏。
- **绝对中立**：通常有个人隐藏目标（存活到特定回合、见证特定事件等）。

## 设计原则

1. 阵营是倾向，不是枷锁：可偶尔违背，但需承受代价。
2. 阵营应被推断，不应直接展示。
3. 阵营只影响白天，不影响夜晚。
4. 单次阵营修正控制在 ±1 ~ ±2，避免覆盖属性与骰子的作用。
