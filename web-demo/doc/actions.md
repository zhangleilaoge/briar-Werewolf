# 白天行动

本文档记录白天阶段所有可用行动的详细规则，包括普通行动与追加行动。

---

## 行动类型

- **普通行动**：消耗玩家自己回合的行动。
- **追加行动**：在触发行动后响应执行，不消耗回合。详见「追加行动」章节。

---

## 普通行动一览

| 行动 | 主要属性 | 说明 |
|------|----------|------|
| 沉默 | 无 | 跳过本回合。全员连续沉默时进入投票。 |
| 公布身份 | 诡诈 / 亲和 | 声明一个本局可能出现的身份，可真可假。 |
| 观察 | 洞察 / 隐蔽 | 暗中观察目标，获取状态与意图提示。 |
| 怀疑 | 洞察 / 逻辑 | 公开表达对某玩家的怀疑。 |
| 袒护 | 亲和 | 公开为某玩家辩护。 |
| 感谢 | 亲和 | 感谢某玩家此前的帮助或袒护。 |
| 号召投票 | 领导 / 逻辑 | 建议大家当天投票给某目标。 |
| 阻止投票 | 领导 / 亲和 | 建议大家当天不要投票给某目标。 |
| 担保清白 | 亲和 / 洞察 | 强烈担保某目标一定是好人。 |
| 强烈指认 | 洞察 / 逻辑 | 强烈指认某目标一定是狼人。 |
| 全员排除 | 逻辑 / 领导 | 某身份有多人自称时，提议全部放逐这些自称者。 |

---

## 沉默

- 跳过本回合，不触发任何效果。
- 连续沉默计数 +1。
- 当连续沉默人数等于存活人数时进入投票。

---

## 公布身份

- 声明一个本局可能出现的身份，可真可假。
- 身份被后续证实时，声明者获得信任/友好提升；被证伪时受到惩罚。
- 撒谎时：己方诡诈对抗其他玩家的逻辑/洞察。
- 说真话时：可用亲和/逻辑增强说服力。
- 限制：每回合一次，不改变实际身份。

---

## 观察

- 选择一名其他玩家暗中观察，获取其状态与意图的提示。
- 提示类型：微表情、言行一致性、情绪状态、阵营/意图倾向、关系线索。均为暗示，非确定性结论。
- 自动触发：目标发动高风险行动（公布身份、担保清白、强烈指认、全员排除）或被怀疑/强烈指认时，系统按观察者洞察自动触发观察。自动观察效果较弱，不触发被发现惩罚。
- 检定：
  - 观察成功：洞察 + d20 vs 目标隐蔽 + d20。
  - 是否被发现：隐蔽 + d20 vs 目标洞察 + d20。
- 限制：每回合一次，不能观察自己，不触发追加行动窗口。

---

## 怀疑

- 公开表达对一名其他玩家的怀疑。
- 效果：目标对怀疑者的信任/友好下降，目标压力 +1~+2；其他玩家记录该怀疑。
- 后续验证：目标被证实为狼人时怀疑者获信任提升；被证实无辜时怀疑者受惩罚。
- 检定：洞察/逻辑 + d20 vs 目标隐蔽 + d20。
- 限制：每回合一次，不能怀疑自己。

---

## 袒护

- 公开为一名其他玩家辩护。
- 效果：目标对袒护者的信任/友好上升，目标压力 -1；其他玩家可能视二者为同盟。
- 后续验证：目标被证实为好人时袒护者获信任提升；被证实为狼人时袒护者受惩罚。
- 检定：亲和 + d20。
- 限制：每回合一次，不能袒护自己。

---

## 感谢

- 公开感谢一名其他玩家此前的帮助、袒护或信息共享。
- 效果：被感谢者对感谢者的友好/信任小幅上升，被感谢者压力 -1~+0。
- 限制：每回合一次，不能感谢自己。

---

## 号召投票

- 公开建议大家当天投票给一名目标。
- 效果：目标对号召者的信任/友好下降，目标压力 +1~+2；提高其他玩家当天投给该目标的概率。
- 检定：领导/逻辑 + d20。
- 限制：每回合一次，不能号召自己。

---

## 阻止投票

- 公开建议大家当天不要投票给一名目标。
- 效果：目标对阻止者的信任/友好上升，目标压力 -1~-2；降低其他玩家当天投给该目标的概率。
- 检定：领导/亲和 + d20。
- 限制：每回合一次，不能阻止自己。

---

## 担保清白

- 强烈担保一名目标一定是好人/村民。
- 效果：目标对担保者的信任/友好大幅上升，目标压力 -1~-2；显著降低目标被投票放逐的概率。
- 后续验证：目标被证实为好人时担保者获大量信任；被证实为狼人时担保者受严重惩罚。
- 检定：亲和/洞察 + d20。
- 限制：每回合一次，不能担保自己，同一目标每天限一次。

---

## 强烈指认

- 强烈指认一名目标一定是狼人/敌人。
- 效果：目标对指认者的信任/友好大幅下降，目标压力 +2~+4；显著提高目标被投票放逐的概率。
- 后续验证：目标被证实为狼人时指认者获大量信任；被证实为好人时指认者受严重惩罚。
- 检定：洞察/逻辑 + d20。目标可在追加窗口内反驳。
- 限制：每回合一次，不能指认自己，同一目标每天限一次。

---

## 全员排除

- 当某身份有两人以上自称时，提议将所有该身份自称者放逐。
- 效果：所有被提议者对提议者的信任/友好大幅下降，被提议者压力 +2~+3；其他玩家评估该提议合理性。
- 检定：逻辑/领导 + d20。
- 限制：仅身份冲突时可发动，每轮白天一次，提议者不能是冲突身份自称者。

---

## 追加行动

部分普通行动会触发追加行动窗口，其他玩家可在不消耗回合的情况下响应。详见 [extra-actions.md](extra-actions.md)。

普通行动中可触发追加行动的有：

| 触发行动 | 追加行动 |
|----------|----------|
| 怀疑 | 一同怀疑 |
| 怀疑 / 一同怀疑 | 反驳 |
| 袒护 | 一同袒护 |
