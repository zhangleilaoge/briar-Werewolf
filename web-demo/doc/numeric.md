# 数值设计

本文档记录游戏中所有数值细节，包括角色属性、角色关系、检定规则与变化幅度。

## 角色属性

### 属性范围

每个角色拥有五项核心属性，数值通常在 **1–10** 之间，可随成长、状态或道具变化。

**重要限制：所有属性只影响白天的投票与讨论阶段，不影响晚上的杀戮阶段。**

### 属性列表

| 属性 | 英文 | 白天阶段作用 |
|------|------|--------------|
| 亲和 | Affinity | 社交、说服、建立信任与联盟 |
| 逻辑 | Logic | 推理、分析、识破谎言与获取信息 |
| 领导 | Leadership | 主导讨论、影响投票、组织团队行动 |
| 诡诈 | Deception | 伪装、欺骗、误导、陷害 |
| 隐蔽 | Stealth | 隐藏真实立场、降低白天被怀疑概率 |

### 属性对白天阶段的影响

#### 亲和（Affinity）

- **说服投票**：说服他人改变投票意向时，亲和提供加值。
- **建立信任**：白天讨论中更容易建立联盟，降低被怀疑的概率。
- **人缘缓冲**：被投票放逐时，亲和较高的角色可能获得额外辩护机会或减票。
- **信息获取**：白天其他角色更愿意主动向高亲和角色分享信息。

#### 逻辑（Logic）

- **线索解读**：白天从公开线索、事件记录中推断信息时，逻辑决定能获取多少有效情报。
- **识破谎言**：对抗白天讨论中目标的诡诈时，逻辑越高越容易发现矛盾或破绽。
- **推理可信度**：白天提出推理时，逻辑影响其他角色是否愿意采信。

#### 领导（Leadership）

- **投票影响**：白天投票阶段，高领导角色发言后，其他角色跟票概率提升。
- **主导议程**：白天讨论阶段，高领导角色拥有更高的发言权重、优先发言或总结权。
- **团队行动**：白天组织多人公开行动时，领导决定协调效率与额外加成。

#### 诡诈（Deception）

- **谎言信服度**：白天撒谎或伪装身份时，诡诈决定谎言被识破的难度。
- **误导投票**：白天将嫌疑转移给他人、引导投票方向的成功率与诡诈相关。
- **假线索**：白天制造或传播假线索、假证据时，诡诈影响可信度。
- **身份伪装**：白天长期伪装成其他阵营或角色时，诡诈决定暴露风险。

#### 隐蔽（Stealth）

- **隐藏立场**：白天隐藏自己的真实阵营与投票倾向，避免被他人察觉。
- **降低怀疑**：白天被针对或盘问时，高隐蔽可降低被怀疑的概率。
- **隐藏痕迹**：白天私下的交流或行动痕迹更难被其他角色发现。

## 角色关系

### 关系规模

每个角色对之间存在**有向关系**，n 名玩家共产生 `n × (n - 1)` 条关系。
每条关系包含两个数值：**信任值**与**友好值**。

### 关系数值范围

| 数值 | 英文 | 含义 | 初始范围 |
|------|------|------|----------|
| 信任值 | Trust | 对对方可靠性与诚实的理性判断 | -10 ~ +10 |
| 友好值 | Friendliness | 对对方的情感倾向与好感 | -10 ~ +10 |

说明：

- 关系是**有向的**：A 对 B 的信任值不等于 B 对 A 的信任值。
- 信任与友好**相互独立**：可能出现「高信任但低友好」或「高友好但低信任」的组合。

### 关系对白天阶段的影响

#### 信任值（Trust）

- **相信言论**：对方发言时，高信任降低识破谎言的意愿；低信任更容易触发怀疑。
- **信息共享**：只有信任值达到阈值，角色才会向对方分享敏感信息。
- **跟票倾向**：高信任对象发起投票号召时，跟票概率提升。
- **辩护意愿**：被投票时，高信任对象更可能为其辩护或减票。
- **说服修正**：试图说服对方时，对方对自己的信任值会修正检定难度。
  - 高信任：说服难度降低
  - 低信任：说服难度增加

#### 友好值（Friendliness）

- **帮助意愿**：白天行动中，高友好角色更愿意主动帮助或合作。
- **保护倾向**：投票时，高友好对象被怀疑，角色更倾向于保护而非跟票。
- **结盟基础**：双向高友好更容易形成非正式联盟。
- **情绪反应**：友好值影响被背叛后的反应强度（友好越高，背叛时信任下降越剧烈）。
- **信任加成**：友好值较高时，信任值的自然恢复与正向提升更快。

### 关系变化

白天事件会改变角色之间的关系值：

| 事件 | 信任值变化 | 友好值变化 |
|------|------------|------------|
| 被证实说真话 / 共同正确推理 | +1 ~ +3 | +0 ~ +1 |
| 被对方帮助 / 辩护 | +1 ~ +2 | +1 ~ +3 |
| 投票一致 | +0 ~ +1 | +0 ~ +1 |
| 被怀疑但未证实 | -1 ~ +0 | -1 ~ +0 |
| 被证实撒谎 / 误导 | -2 ~ -5 | -1 ~ -3 |
| 背叛联盟 / 背后投票 | -3 ~ -6 | -2 ~ -5 |
| 被投票放逐（未成功） | -1 ~ -3 | -1 ~ -2 |

每天关系值变化设有上限，避免极端波动。

### 与属性的互动

- **亲和**：亲和高的角色更容易提升他人对自己的友好值。
- **逻辑**：逻辑高的角色在提供正确推理时更容易获得信任值。
- **领导**：领导高的角色号召投票时，信任值会放大跟票效果。
- **诡诈**：诡诈高的角色撒谎更不容易降低信任值（即使被发现，惩罚较小）。
- **隐蔽**：隐蔽高的角色在隐藏真实立场时，不容易因立场暴露而降低关系值。

## 检定规则

### 基本检定

白天阶段的一般行动采用「属性 + 骰子」的检定方式：

```
检定结果 = 对应属性值 + d20
难度 = 目标对应属性值 或 固定难度值（如 15）
```

### 成功等级

| 结果 | 条件 | 效果 |
|------|------|------|
| 大失败 | 检定结果 ≤ 难度 - 10 | 行动失败，并可能触发负面后果 |
| 失败 | 检定结果 < 难度 | 行动未达成预期效果 |
| 成功 | 检定结果 ≥ 难度 | 行动达成预期效果 |
| 大成功 | 检定结果 ≥ 难度 + 10 | 行动获得额外收益或隐藏信息 |

### 对抗检定

部分白天行动由双方属性对抗决定：

| 行动 | 检定公式 |
|------|----------|
| 说服投票 | 己方亲和 + d20 vs 对方逻辑 + d20 |
| 识破谎言 | 己方逻辑 + d20 vs 对方诡诈 + d20 |
| 隐藏立场 | 己方隐蔽 + d20 vs 对方逻辑 + d20 |

### 含关系修正的检定

白天说服检定的完整公式：

```
说服难度 = 基础难度 + 对方逻辑 - 对方对自己的信任值

己方检定 = 己方亲和 + d20
```

如果检定成功，可能进一步改变双方关系值（友好值小幅上升，信任值根据内容上升或不变）。
