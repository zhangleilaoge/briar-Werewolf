# 道具

## 基本概念

- **道具**是角色拥有的物品，可以启用特定行动、提供检定加值或改变某些规则。
- 道具与属性、特质不同：属性是角色固有能力，特质是角色特性规则，而道具是外部物品，可能获得、失去或转移。
- 所有道具默认为**隐藏道具**，其他玩家需通过观察或特殊能力才能发现。
- 所有道具都有**两种功效**：根据持有者的阵营（狼人阵营 / 人类阵营）触发不同效果。
- 每个角色最多拥有 **3 个道具栏位**。获得新道具时若栏位已满，需先丢弃或转移一个已有道具。

## 道具类型

| 类型 | 说明 |
|------|------|
| 行动前提型 | 拥有该道具才能执行特定行动 |
| 检定加值型 | 在特定检定中提供加值或减值 |
| 一次性消耗型 | 使用一次后消失 |
| 被动效果型 | 持续生效，无需主动使用 |

## 道具示例

| 道具 | 类型 | 狼人阵营功效 | 人类阵营功效 |
|------|------|--------------|--------------|
| 尖牙利爪 | 行动前提型 | 拥有时可在夜晚执行一对一杀戮 | 被攻击时可选择与攻击者同归于尽 |
| 水晶球 | 行动前提型 | 无效果 | 预言家持有时可执行夜间查验；若查验到狼人，水晶球碎裂损坏 |
| 小偷手套 | 行动前提型 | 无效果 | 窃贼持有时可执行一次偷取；使用后损坏 |
| 验尸工具 | 一次性消耗型 | 无效果 | 验尸官持有时可执行一次尸检，查看一名死亡角色的所有道具；使用后损坏 |
| 护身符 | 一次性消耗型 | 抵挡一次夜晚杀戮，使用后损坏 | 抵挡一次夜晚杀戮，使用后损坏 |

> 其他道具将在后续设计中加入。

## 道具与角色

- 某些道具是角色的默认装备，用于区分角色能力。
- 例如：普通狼人默认拥有「尖牙利爪」，失去该道具后将无法执行夜晚杀戮，但**仍是狼人阵营**。
- 道具效果随持有者阵营变化：同一道具在狼人手中与人类手中可能触发不同功效。
- 道具的获得与失去可以由剧情、角色能力、玩家行动或随机事件触发。
- 一般而言，失去道具不会直接改变角色的身份或阵营，除非有特殊规则另行说明。

## 道具与行动

- 若某行动需要特定道具作为前提，角色没有该道具时无法执行该行动。
- 道具的检定加值可以与属性、特质、阵营修正叠加，但需遵循数值上限规则。
