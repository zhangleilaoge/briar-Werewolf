# 特质

## 基本概念

- **特质（Trait）**是角色拥有的特殊效果或规则修正。
- 所有非基础属性产生的人物效果，都归类于特质。
- 特质可以影响阵营倾向、属性生效方式、压力值变化、关系变化、行动效果等。

## 与属性的区别

| 类型 | 性质 | 例子 |
|------|------|------|
| 属性 | 六维基础数值，直接参与检定 | 亲和、逻辑、领导、诡诈、隐蔽、洞察 |
| 特质 | 特殊规则或修正，改变数值的运行方式 | 混乱倾向、胆小、冷静、煽动者 |

- 属性决定检定的基础加值；特质决定某些情境下数值如何被修正或特殊规则如何触发。

## 特质的来源

- **预设角色**：部分预设角色自带固定特质，用于体现其性格与背景。例如爱蜜莉雅可能拥有「混乱倾向」特质。
- **随机角色**：开局时可能随机获得少量特质，增加变数。
- **剧情模式**：特定剧情线可能临时赋予或移除角色特质。

## 特质示例

以下是部分特质的设计示例，实际游戏中可扩展：

| 特质 | 效果 |
|------|------|
| 混乱倾向 | 阵营倾向永远落在混乱区域，不会在守序或中立区域出现。 |
| 胆小 | 压力值增加速率是常人的两倍，压力值自然恢复速度减半。 |
| 冷静 | 压力值增加速率减半，高压力时对诡诈/隐蔽检定的减值降低。 |
| 煽动者 | 号召投票与强烈指认的说服力额外提升，但失败时信任惩罚加重。 |
| 固执 | 被说服改变投票意向的难度更高，但自身跟票倾向也较低。 |
| 共情 | 降低他人压力的效果增强，更容易通过袒护/感谢提升友好值。 |
| 多疑 | 更容易自动观察他人，但对观察结果的误判率略高。 |
| 忠诚 | 对高信任对象的背叛惩罚更重，但跟票与袒护意愿更强。 |

## 与预设角色的关系

- 预设角色通常拥有 1~3 个固定特质，用于区分其与随机角色的差异。
- 特质体现角色的性格基调，但不锁定其阵营或身份。
- 例如：爱蜜莉雅拥有「混乱倾向」特质，因此她的阵营每局都会在混乱善良、混乱中立、混乱邪恶中浮动，不会跳出混乱区域。

## 与随机角色的关系

- 随机角色也可能获得特质，但通常是少量、随机分配的。
- 随机角色的特质组合更不稳定，可能产生意外的人格特征。

## 设计原则

1. **特质是规则修正，不是属性替代**：特质改变数值的运行方式，不直接替代六维属性。
2. **特质应有正有负**：强力正面效果通常伴随代价或限制。
3. **特质可被观察与推断**：其他角色可以通过言行和观察，逐步推测某角色可能拥有的特质。
4. **特质与阵营可分离**：特质影响阵营倾向，但不等同于阵营；角色仍可能在特质倾向范围内出现不同阵营。
